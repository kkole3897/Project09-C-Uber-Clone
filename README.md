## 프로젝트 소개

✅  Google Maps API 를 이용한 위치 조회, 장소 검색, 경로 조회 

✅  택시 기사와 사용자 간의 실시간 통신을 통한 호출, 응답, 탑승, 하차 알림

✅  매칭된 택시 기사와 사용자 간의 실시간 위치 공유

✅  모바일 기반의 웹 서비스

## 🚀 How to start

```bash
# client
cd client
npm install
npm start
```

```bash
# server
cd server
npm install
npm run dev
```
